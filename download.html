MacOS support is experimental because the game is not signed.

You can still play the game, but the first start requires additional steps.

Download the .zip file and extract the app from it. Before starting the app, open System Preferences, and go to “Security & Privacy”, open the “General” tab and unlock it using the lock at the bottom.

Now you can launch the oxrox app. It may take up to 5 minutes. MacOS will complain that it is not from a known developer.

Press “cancel” and go to the system preferences window. It will contain a button “run anyway” to run the game.

The game should now start. All further starts will not require this procedure.

System Requirements

  • 64 bit Operating System
  • OpenGL ES 3.0 capable graphics card
  • 2 GB system memory
  • 200 MB disk space
